Question-1: What are the different ways to select an element in the DOM?

We can select elements using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. These methods allow us to target elements by their ID, class, tag name, or CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns or sets the HTML content inside an element, including tags. innerText returns only the visible text, ignoring hidden elements and styles. textContent returns all text content, including hidden elements, but excludes HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we attach an event listener to a parent element to handle events for its child elements. This is useful for dynamically added elements or to improve performance by reducing the number of event listeners.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event propagates from the target element up through its ancestors in the DOM tree. This allows parent elements to handle events triggered by their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: Use document.createElement('tagName').
Add: Use appendChild() or insertBefore() to add elements to the DOM.
Remove: Use removeChild() or remove() to delete elements from the DOM.
